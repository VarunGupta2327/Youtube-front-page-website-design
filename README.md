🎬 YouTube Front Page Website Design

This project is a YouTube Front Page Clone created using HTML and CSS.
It replicates the layout and structure of YouTube’s homepage including:

Header with logo and search bar

Sidebar navigation menu

Video thumbnails grid layout

Responsive design

This project is made for learning and practice purposes to improve frontend development skills.

🚀 Features

✨ Clean and modern UI
📱 Fully Responsive Design
🎨 Styled using pure CSS
📂 Organized folder structure
🖥️ Desktop and Mobile friendly layout

🛠️ Technologies Used

HTML5

CSS3

Flexbox

CSS Grid
